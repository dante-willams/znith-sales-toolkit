Critical Pattern Changes — v1 → v2
These are the specific patterns Claude Code must apply. Each one replaces a v1 pattern. Read these carefully before running the update prompt.
/* ── HEADER: was transparent/dark, now navy ── */ header, .header, [class*="header"] { background: var(--navy); /* was: rgba(14,13,11,0.95) or #181714 */ border-bottom: 1px solid rgba(255,255,255,0.06); backdrop-filter: none; /* remove blur — not needed on solid navy */ } /* Wordmark text inside header stays white/amber — no change needed */
/* ── BODY + PAGE BACKGROUND ── */ body { background: var(--bg); /* was: #060503 or #0e0d0b */ color: var(--text); /* was: rgba(255,255,255,0.85) */ }
/* ── CARDS, PANELS, MODALS ── */ /* Replace all dark surface colors */ .card, .panel, .stage, .modal, [class*="card"], [class*="panel"] { background: var(--surf); /* was: #181714 */ border: 1px solid var(--border); /* was: rgba(255,255,255,0.07) */ } /* Stage body / elevated areas */ .stage-body, .card-body, [class*="stage-body"] { background: var(--surf-2); /* was: #222018 */ }
/* ── TEXT COLORS — replace all rgba white values ── */ /* Primary text */ color: var(--text); /* replaces: color: rgba(255,255,255,0.85) */ /* Secondary / muted text */ color: var(--tmuted); /* replaces: color: rgba(255,255,255,0.45) */ /* Labels, placeholders, faint copy */ color: var(--tfaint); /* replaces: color: rgba(255,255,255,0.20) */ /* Exception: text INSIDE the navy header/footer stays rgba white */ /* Do not change text colors scoped inside header {} or .znith-footer {} */
/* ── ACTIVE STAGE BLOCK — replace glow with border-left accent ── */ .stage.active, .stage-block.active, [class*="stage"][class*="active"] { background: var(--surf); border: 1px solid var(--border); border-left: 3px solid var(--accent); /* accent = module color */ box-shadow: none; /* remove: was 0 0 0 1px amber + glow */ } /* Stage number chip uses --accent, not --amber (unless Prospecting Agent) */ .stage-num, .step-num { background: var(--accent); color: #000; }
/* ── FOOTER — navy with solid amber top border ── */ .znith-footer, footer { background: var(--navy); /* was: var(--surf) #181714 */ border-top: 2px solid var(--amber); /* was: 1px solid rgba(184,134,11,0.15) */ /* All other footer styles unchanged */ } /* Full footer HTML template (replace existing footer): */ <footer class="znith-footer"> <div class="znith-wordmark"> <span style="font-family:'IBM Plex Mono';color:#B8860B;font-weight:700;">/</span> <span style="font-family:'Epilogue';font-weight:800;color:#B8860B;">Z</span> <span style="font-family:'Epilogue';font-weight:800;color:rgba(255,255,255,0.45);">nith</span> </div> <div style="display:flex;align-items:center;gap:8px;"> <span style="font-family:'IBM Plex Mono';font-size:10px;color:rgba(255,255,255,0.35);">[TOOL NAME]</span> <div style="width:4px;height:4px;border-radius:50%;background:#B8860B;"></div> <span style="font-family:'IBM Plex Mono';font-size:10px;color:rgba(255,255,255,0.35);">[VERSION]</span> </div> </footer>

Do not change any functionality, API calls, prompt logic, data structures, or JavaScript. Set stage-body / elevated areas to background: var(--surf-2). STEP 5 — Text colors Replace rgba(255,255,255,0.85) → var(--text) Replace rgba(255,255,255,0.45) → var(--tmuted) Replace rgba(255,255,255,0.20) → var(--tfaint) Exception: do NOT change text colors inside header {} or footer {}. Those elements are navy background — keep their rgba white values. STEP 6 — Active stage block Remove: border glow (box-shadow: 0 0 0 1px ..., 0 4px 20px ...)
